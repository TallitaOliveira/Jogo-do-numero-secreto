# 🔢 Jogo do Número Secreto

Jogo interativo onde o utilizador tenta adivinhar um número secreto gerado aleatoriamente pelo sistema.

## 🎮 Como jogar

1. O sistema gera um número secreto aleatório
2. O utilizador digita um palpite
3. O sistema diz se o número é maior ou menor
4. O objetivo é acertar no menor número de tentativas possível!
